# SDL_PingPong
PingPong singleplayer game on C++ and SDL3.

Uses SDL3, SDL3_ttf and SDL3_mixer.

### On Windows:

Before compile, download this libs and extract to some folder.

Default paths on this project is:

```
C:\Dev\SDL\MinGW_SDL3
C:\Dev\SDL\MinGW_SDL3_ttf
C:\Dev\SDL\MinGW_SDL3_mixer
```

Default MinGW path is:

```
C:\Dev\MinGW\mingw64
```

from https://github.com/niXman/mingw-builds-binaries/releases/tag/15.2.0-rt_v13-rev0

To build, open folder in VS Code, then press Ctrl + Shift + B and choose build task from list.

### On Linux:

Just install libs SDL3, SDL3_ttf(In some distro's SDL3-ttf) and SDL3_mixer.

You can use Makefile or from terminal by g++ :)

# SDL_PingPong

Одиночная игра Пинг-понг. Написана на C++ с графической библиотекой SDL3.

Использованы библиотеки: SDL3, SDL3_ttf and SDL3_mixer.

### На Windows:

Перед компиляцией, скачайте сами библиотеки и распакуйте в какую-нибудь папку.

По умолчанию, в этом проекте пути к ним:
```
C:\Dev\SDL\MinGW_SDL3
C:\Dev\SDL\MinGW_SDL3_ttf
C:\Dev\SDL\MinGW_SDL3_mixer
```

Путь к MinGW:
```
C:\Dev\MinGW\mingw64
```

Сборка https://github.com/niXman/mingw-builds-binaries/releases/tag/15.2.0-rt_v13-rev0

Для сборки, открыть папку в VS Code, затем нажать комбинацию Ctrl + Shift + B и выбрать задачу сборки из списка.

### На Linux:

Просто установите библиотеки SDL3, SDL3_ttf(В некоторых дистрибутивах SDL3-ttf) и SDL3_mixer.

Вы можете использовать Makefile или просто скомпилировать из терминанала командой g++ :)
